# Overriding과 Overloading

### Overriding

* 하위 클래스에서 상위 클래스의 메서드를 재정의 하는 것.
* 쉽게 말해 메서드 이름이 같고, 매개변수가 같고, 반환형 타입이 같은 상위 클래스의 메서드를 하위 클래스에서 덮어쓴다고 생각하면 된다.



### Overloading

* 두 메서드가 같은 이름을 갖고 있지만, 매개변수의 개수나 자료형이 다르게 하여 메서드를 여러 개 정의하는 것.
